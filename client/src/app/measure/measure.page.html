NASA Task Load Index History

The NASA Task Load Index (NASA-TLX) is a multi-dimensional assessment tool that rates perceived workload in order to assess a task, system, or team's effectiveness. Developed by NASA's Human Performance Group in the 1980s, this tool has been used in various fields to assess job demands and design, operator training needs, and system usability.

 Questions

The NASA-TLX comprises six subscales, each addressing a different aspect of perceived workload. These include Mental Demand, Physical Demand, Temporal Demand, Performance, Effort, and Frustration.

 Scoring

Each subscale is rated on a scale of 0 (very low) to 100 (very high). The scores of these subscales can be used separately or combined into a global workload score, which provides a comprehensive view of the overall workload.

 How It Works

NASA-TLX provides a graphical representation of workload, whereby higher scores indicate greater workload. This tool helps researchers and designers identify tasks that may be too demanding or aspects of a task that could be redesigned to reduce workload.
